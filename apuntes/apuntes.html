APUNTES DE DJANGO



Iniciar proyecto

django-admin startproject nombre_proyecto



Iniciar servidor

En el mismo directorio que el archivo "manage.py"

python manage.py runserver



Aplicar migraciones de apps iniciales en BD

En el mismo directorio que el archivo "manage.py"

python manage.py migrate



Crear app

En el mismo directorio que el archivo "manage.py"

python manage.py startapp nombre_de_la_app

from django.contrib import admin from django.urls import path from core import views urlpatterns = [ path('admin/', admin.site.urls), path('', views.home, name="home"),
]
